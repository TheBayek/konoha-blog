L'AKATSUKI

PRÉSENTATION

L'Akatsuki est une organisation criminelle constituée de ninjas déserteurs (nukenin) de rang S qui sont presque tous représentés dans le Bingo Book.

MEMBRES

  • Itachi Uchiha
  • Madara Uchiha
  • Sasori
  • Kakuzu
  • Hidan
  • Deidara
  • Zetsu
  • Kisame Hoshigaki

CHEFS

  • Yahiko (Fondateur)
  • Pain / Nagato
  • Obito Uchiha (Tobi)

OBJECTIF

L'objectif de l'Akatsuki était de base, dans l'Akatsuki originel, de pacifier le vilage d'Ame pacifquement. Mais depuis la mort de Yahiko, l'organisation a changé de direction. L'objectif de l'akatsuki actuel (tobi/Pain) est de captuer les neufs Bijûs pour "pacifier le monde" selon Tobi, pour envoyer la population modiale dans une illusion où toutes les personnes vivraient comme ils le veulent.
